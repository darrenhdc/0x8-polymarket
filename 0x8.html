Overview

Autonomous AI agent for trading on Polymarket prediction markets. Inspired by a friend's success with OpenClaw (where an agent turned $5000 virtual funds into $5090 in one night), this project aims to build a similar system using Claude Code as the foundation.

The agent will independently design trading strategies, monitor Polymarket markets, and make trading decisions without manual intervention. The goal is to explore the capabilities of AI in prediction market trading while maintaining proper risk management.

Architecture Decision: Claude Code vs OpenClaw

Recommendation: Claude Code

Why Claude Code is preferred:

  • Full control & transparency - we own the entire codebase
  • Already in this environment - seamless integration
  • MCP Tools available - web search, data fetching, etc.
  • Iterative development - quickly test and modify strategies
  • No black box - understand every decision the agent makes

OpenClaw Consideration

OpenClaw is impressive and proven, but as a black-box system it doesn't align with the lab's research goals of understanding and iterating on every component of the trading system.
