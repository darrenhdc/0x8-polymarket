🧠 完整策略文档

✅ 同时支持 YES 和 NO 结果

系统分析并可以交易任何市场的 YES 和 NO 结果。策略引擎独立评估双方，选择提供最佳风险/回报的一方。

  • • 所有 5 种策略都可以触发 BUY_YES 或 BUY_NO
  • • 当前只有 YES 持仓是因为入场时的市场机会
  • • 只要出现好的机会，NO 持仓会自动添加

📊 5 种活跃策略

1. 动量策略

购买高流动性的低价格结果（10-35%）。可以触发 BUY_YES 或 BUY_NO。

2. 非对称价值

专注于 15-35% 概率区间以获得最佳风险/回报。可以触发 BUY_YES 或 BUY_NO。

3. 反向策略

对抗 >90% 的信心，但最低 10% 价格。可以触发 BUY_YES 或 BUY_NO。

4. 错误定价套利

寻找 YES + NO ≠ 1.0 的机会。可以触发 BUY_YES 或 BUY_NO。

5. 流动性套利

寻找订单簿不平衡和紧缩价差。可以触发 BUY_YES 或 BUY_NO。

🛡️ 风险管理规则

  • 动态止损

    低概率 10%，正常市场 15%

  • 止盈

    30% 收益触发自动卖出

  • 24小时冷却期

    止损后防止重新入场

  • 仓位大小

    根据风险等级 $200-$500

  • 最低价格

    10% 以下不做反向操作（从损失中学习）

📚 经验教训

❌ 失败的经验

  • • OpenAI 硬件：买入价 3.9% → 止损 -21.8% 损失
  • • GTA VI：买入价 2.4% → 止损 -17.0% 损失

✅ 成功的经验

  • • 世界杯持仓：稳定，低波动性
  • • 中概率区间：15-35% 有最佳风险/回报
  • • 冷却机制：防止重复损失
  • • 动态止损：有效保护资金
